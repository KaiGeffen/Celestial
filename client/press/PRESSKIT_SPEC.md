# Press Kit Specification

## URL

`celestialdecks.gg/press/`

## Must Include

- Navigation Bar (About, Press, Play Game links)
- Hero Section with logo
- Internal Navigation (anchor links to all sections)
- Factsheet Section
- Description Section
- Features Section
- History Section
- Videos Section
- Images Section (with downloadable screenshots)
- Logos Section (with downloadable logo)
- Credits Section
- Contact Section

## Navigation Bar

- Sticky top navigation
- Links: About, Press, Play Game
- Current page should be highlighted

## Hero Section

- Logo display
- Background image from `../about/assets/background.webp`

## Internal Navigation

- Anchor links to all sections:
  - Factsheet
  - Description
  - Features
  - History
  - Videos
  - Images
  - Logos
  - Credits
  - Contact

## Factsheet Section

**Required fields:**

- Title: Celestial Decks
- Genre: Strategy Card Game
- Platform: Web Browser (PC, Mac), Mobile in development
- Release date: Available now in beta, 1.0 release planned December 1st 2026
- Price: Free to Play
- Website: celestialdecks.gg
- Twitter: @CelestialDecks
- Discord: Join Community (link)
- YouTube: @CelestialDecks

## Description Section

- Short game description (2-3 paragraphs)
- Should explain the core gameplay and unique selling points

## Features Section

- Bulleted list of key features (8 items)
- Should highlight unique mechanics and gameplay elements

## History Section

- Development story (2-3 paragraphs)
- Background on how the game was created
- Launch information

## Videos Section

**Required videos:**

- Gameplay Trailer (YouTube link)

## Images Section

**Required images:**

- Deck Building Screenshot (`feature-deckbuilding.webp`)
- Journey Mode Screenshot (`feature-journey.webp`)
- Competitive Play Screenshot (`feature-compete.webp`)
- Social Preview Image (`social-preview.webp`)

**Image requirements:**

- All images should be displayed in a responsive grid
- Each image should be downloadable
- Images should have hover effects

## Logos Section

**Required logo assets:**

- Full Logo (SVG format) - `logo.svg`
- Logo should be displayed with preview
- Download link for SVG logo

## Credits Section

**Required information:**

- Developer name/team
- Website link

## Contact Section

**Required contact methods:**

- Press Inquiries: press@celestialdecks.gg
- General Inquiries: info@celestialdecks.gg
- Twitter: @CelestialDecks (link)
- Discord: Join our Discord server (link)
